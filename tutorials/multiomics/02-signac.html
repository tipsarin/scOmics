6.1 Overview

Signac extends Seurat to handle chromatin accessibility data. When paired with matched scRNA-seq (e.g., 10x Multiome), it enables joint analysis — linking open chromatin regions to gene expression at single-cell resolution.

This chapter covers:

  1. Creating a Signac object from a 10x Multiome dataset
  2. ATAC-seq QC within Signac
  3. LSI dimensionality reduction
  4. Joint WNN embedding with matched RNA
  5. Linking peaks to genes

6.2 Setup

Code 
library(Seurat)
library(Signac)
library(EnsDb.Hsapiens.v86)
library(BSgenome.Hsapiens.UCSC.hg38)
library(ggplot2)
library(patchwork)

6.3 Load 10x Multiome Data

Code 
# Read Cell Ranger ARC output
counts <- Read10X_h5("data/raw/pbmc_multiome/filtered_feature_bc_matrix.h5")

# Separate RNA and ATAC matrices
rna_counts  <- counts$`Gene Expression`
atac_counts <- counts$Peaks

# Load fragment file metadata
fragpath <- "data/raw/pbmc_multiome/atac_fragments.tsv.gz"

6.4 Create Seurat + Signac Object

Code 
# Genome annotation
annotation <- GetGRangesFromEnsDb(ensdb = EnsDb.Hsapiens.v86)
seqlevels(annotation) <- paste0("chr", seqlevels(annotation))

# Create chromatin assay
atac_assay <- CreateChromatinAssay(
  counts     = atac_counts,
  sep        = c(":", "-"),
  fragments  = fragpath,
  annotation = annotation
)

# Build Seurat object with both modalities
seu <- CreateSeuratObject(counts = rna_counts, assay = "RNA")
seu[["ATAC"]] <- atac_assay

seu

6.5 ATAC QC

Code 
DefaultAssay(seu) <- "ATAC"

seu <- NucleosomeSignal(seu)
seu <- TSSEnrichment(seu, fast = FALSE)

# Filter
seu <- subset(
  seu,
  subset = nCount_ATAC > 1000   &
           nCount_ATAC < 100000 &
           nCount_RNA  > 200    &
           nCount_RNA  < 25000  &
           nucleosome_signal < 2 &
           TSS.enrichment > 1
)

6.7 Joint WNN Embedding

Code 
seu <- FindMultiModalNeighbors(
  seu,
  reduction.list = list("pca", "lsi"),
  dims.list      = list(1:30, 2:50),  # exclude LSI component 1 (correlated with depth)
  modality.weight.name = "RNA.weight"
)

seu <- RunUMAP(
  seu,
  nn.name    = "weighted.nn",
  reduction.name = "wnn.umap",
  reduction.key  = "wnnUMAP_"
)

seu <- FindClusters(seu, graph.name = "wsnn", resolution = 0.8)
Code 
p1 <- DimPlot(seu, reduction = "wnn.umap", label = TRUE) + NoLegend()
p2 <- FeaturePlot(seu, features = "RNA.weight", reduction = "wnn.umap")
p1 | p2

## Overview

[Signac](https://stuartlab.org/signac/) extends Seurat to handle chromatin
accessibility data. When paired with matched scRNA-seq (e.g., 10x Multiome),
it enables joint analysis — linking open chromatin regions to gene expression
at single-cell resolution.

**This chapter covers:**

1. Creating a Signac object from a 10x Multiome dataset
2. ATAC-seq QC within Signac
3. LSI dimensionality reduction
4. Joint WNN embedding with matched RNA
5. Linking peaks to genes

## Joint WNN Embedding

```{r}
#| label: wnn
#| eval: false

seu <- FindMultiModalNeighbors(
  seu,
  reduction.list = list("pca", "lsi"),
  dims.list      = list(1:30, 2:50),  # exclude LSI component 1 (correlated with depth)
  modality.weight.name = "RNA.weight"
)

seu <- RunUMAP(
  seu,
  nn.name    = "weighted.nn",
  reduction.name = "wnn.umap",
  reduction.key  = "wnnUMAP_"
)

seu <- FindClusters(seu, graph.name = "wsnn", resolution = 0.8)
```

```{r}
#| label: wnn-plot
#| eval: false
#| fig-cap: "WNN UMAP colored by cluster and RNA modality weight"

p1 <- DimPlot(seu, reduction = "wnn.umap", label = TRUE) + NoLegend()
p2 <- FeaturePlot(seu, features = "RNA.weight", reduction = "wnn.umap")
p1 | p2
```

## Link Peaks to Genes


