Getting Started

Installation, environment setup, and data download

Prerequisites

These tutorials assume:

  • R ≥ 4.3.0 installed (or access to an HPC with R modules)
  • Familiarity with R syntax and Bioconductor
  • Basic understanding of single-cell library preparation

Environment Management with renv

We strongly recommend using renv to lock package versions per project, ensuring reproducibility across machines and time.

install.packages("renv")
renv::init()       # initialise in your project directory
renv::snapshot()   # lock current package versions to renv.lock
renv::restore()    # restore from renv.lock on another machine

Install Core Packages

Seurat + Harmony + Monocle3

# Seurat v5
install.packages("Seurat")
install.packages("SeuratData")

# Harmony
install.packages("harmony")

# Monocle3 dependencies (Bioconductor)
if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install(c(
  "BiocGenerics", "DelayedArray", "DelayedMatrixStats",
  "limma", "lme4", "S4Vectors", "SingleCellExperiment",
  "SummarizedExperiment", "batchelor", "HDF5Array",
  "terra", "ggrastr"
))

devtools::install_github("cole-trapnell-lab/monocle3")

ArchR (scATAC-seq)

devtools::install_github(
  "GreenleafLab/ArchR",
  ref   = "master",
  repos = BiocManager::repositories()
)
ArchR::installExtraPackages()
Note

ArchR installation requires devtools. On HPC systems, you may need to load a git module before running devtools::install_github().

Signac (scATAC / Multi-omics)

BiocManager::install(c(
  "BSgenome.Hsapiens.UCSC.hg38",
  "EnsDb.Hsapiens.v86",
  "TFBSTools",
  "JASPAR2020"
))
install.packages("Signac")

Visualization

BiocManager::install("ComplexHeatmap")
install.packages(c(
  "ggplot2", "patchwork", "ggrepel",
  "viridis", "RColorBrewer", "scales",
  "cowplot", "ggsci"
))

Running on HPC

If you are working on an HPC cluster (as most of these workflows require), load modules before starting R:

module load R/4.3.3
module load gcc/11.2.0    # required for some package compilation
module load hdf5           # required for ArchR / HDF5Array

For memory-intensive steps (e.g., ArchR Arrow file creation, Harmony integration), request adequate resources:

# Example SLURM header
#SBATCH --mem=64G
#SBATCH --cpus-per-task=16
#SBATCH --time=04:00:00
