Tutorials are developed and tested with the following versions:

Package Version Purpose
R ≥ 4.3.0 Base environment
Seurat 5.x scRNA-seq, Spatial, WNN
ArchR 1.0.3 scATAC-seq
Signac 1.13.0 Chromatin accessibility
Monocle3 1.3.x Trajectory analysis
Harmony 1.2.x Batch integration
ComplexHeatmap 2.18.x Visualization
ggplot2 3.5.x Visualization

0.3 How to Use These Tutorials

Each chapter follows a consistent structure:

  1. Overview — biological rationale and goals
  2. Setup — required packages and data
  3. Code walkthrough — annotated, reproducible R code
  4. Parameter guidance — when and how to adjust defaults
  5. Checkpoints — saveRDS() calls to preserve progress
  6. Session info — exact package versions used
